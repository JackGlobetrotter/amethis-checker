Inscription sur la liste de diffusion Amethis-Checker Vous pouvez vous inscrire par le formulaire ci-dessus sur une liste de diffusion qui vous permet de recevoir des emails dès lors qu'une nouvelle formation a été ajoutée sur Amethis. De plus, il existe une fonctionnalité pour recevoir des rappels d'inscription lorsque cette dernière n'est pas encore possible au moment de l'envoi du mail.

Afin d'ajouter ou supprimer votre adresse mail de la liste de diffusion, veuillez la rentrer dans le champ ci-dessous. Le mot de passe vous a été transmis dans le mail d'information.



Merci de renseigner le mot de passe fourni dans le mail d'information.
